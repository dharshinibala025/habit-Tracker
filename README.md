# HabitFlow

A modern, minimalist habit tracker built with React.

## Features

- **Daily Tracking**: Mark habits as complete day by day.
- **Streaks**: Visualize your consistency with streak counters.
- **Progress Overview**: View daily completion percentage and active habits.
- **Local Persistence**: Data is saved automatically in your browser's local storage.
- **Light Theme**: Clean, pastel aesthetic for a calm user experience.

## Tech Stack

- React
- Vite
- Vanilla CSS (Custom Design System)
- `date-fns` for date manipulation
- `lucide-react` for icons

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open your browser at `http://localhost:5173`.
